    👥 Ученики

    Раздел «Ученики» — основной центр управления учебным процессом. Здесь можно добавлять новых учеников, управлять их данными, отслеживать успеваемость и настраивать индивидуальные параметры обучения.

    🎷 Типы учеников в системе

    Платформа разделяет учеников на два типа, что позволяет гибко вести учёт даже тех, кто не пользуется Telegram:

    1. Реальный ученик — зарегистрирован в боте KITON.
      • Доступен полный функционал: отправка домашних заданий, автоматические уведомления, система «Газиков» и подключение родителей.
    2. Виртуальный ученик — профиль, созданный репетитором вручную.
      • Используется для ведения расписания, учёта оплат и статистики.
      • Нет связи с Telegram-аккаунтом, поэтому ученик не получает уведомления и задания через бота.
      • Виртуального ученика можно в любой момент превратить в реального — вся история занятий и оплат сохранится.

    📋 Список учеников и индикаторы

    В главном списке ученики группируются по предметам. Для быстрой ориентации используются статусы-эмодзи:

    • 📝 — у ученика есть активное домашнее задание.
    • 📤 — ученик сдал ДЗ, но оно ещё не проверено (нет оценки).
    • ✅ — ДЗ проверено и оценено.
    • 🆕 — у ученика нет домашних заданий.
    • 👤 — реальный ученик (отображается, если нет ДЗ или статус не определён).
    • 👻 — виртуальный ученик.
    • 🔒 — ученик заблокирован из-за ограничений тарифа (нужно повысить тариф для доступа).

    Если учеников больше 15, список разбивается на страницы с кнопками навигации ◀️ и ▶️.

    ➕ Добавление и подключение учеников

    Добавление выполняется через меню Ученики → ✏️ Редактировать → ➕ Добавить ученика. Доступно три способа:

    1. 🔗 По ссылке (рекомендуется)

    Самый быстрый способ.

    1. Бот создаёт виртуальный профиль и выдаёт уникальную ссылку.
    2. Отправьте эту ссылку ученику.
    3. Как только ученик перейдёт по ссылке и запустит бота, его профиль автоматически свяжется с вашим, а статус сменится с «виртуального» на «реальный».

    Никаких кодов вводить не нужно — подтверждение происходит автоматически.

    2. 🔑 По персональному коду

    Подходит, если ученик уже зарегистрирован в боте.

    1. Попросите ученика отправить персональный код (например, ABCD-1234-EFGH). Код находится в разделе 👤 Профиль.
    2. В меню выберите Ученики → ✏️ Редактировать → ➕ Добавить ученика → По коду.
    3. Введите код — ученик подключится сразу как реальный.

    3. 👻 Виртуальный ученик

    Используется, если ученик ещё не зарегистрирован или вы хотите вести учёт независимо.

    1. В меню выберите Ученики → ✏️ Редактировать → ➕ Добавить ученика → Виртуальный ученик.
    2. Введите имя и фамилию ученика.
    3. Можно сразу вести занятия, задания, оплаты и статистику.
    4. Когда ученик зарегистрируется в боте, подключите его к виртуальному профилю (кнопка 🔗 Подключить реального ученика в карточке ученика).

    🔄 Восстановление учеников

    Если ученик был удалён (архивирован), его можно восстановить:

    1. Перейдите в Ученики → ✏️ Редактировать.
    2. Нажмите 🔄 Восстановить ученика.
    3. Выберите ученика из списка архивных.

    Вся история занятий и оплат сохраняется при восстановлении.

    📄 Карточка ученика

    При нажатии на ученика в списке открывается его карточка. Она содержит:

    Для реального ученика:

    • 👤 Имя и фамилия (ссылка на Telegram-профиль)
    • 🏫 Школа, класс, город
    • 📞 Телефон
    • 🎮 Количество накопленных Газиков
    • 📚 Список предметов с почасовой ставкой и статусом ссылок (Доска, Созвон)
    • 📅 Расписание ближайших занятий (регулярных и разовых)

    Для виртуального ученика:

    • 👻 Имя и фамилия (виртуальный)
    • 🔑 Персональный код для подключения
    • 🏫 Школа, класс, город (если указаны)
    • 📞 Телефон (если указан)

    Кнопки карточки ученика

    Кнопка Описание
    📊 Успеваемость Статистика по ученику: занятия, ДЗ, Газики
    📅 Слот занятия Нажмите на занятие в расписании, чтобы перейти к его редактированию
    ➕ Добавить занятие Создать новое регулярное или разовое занятие
    ✏️ Редактировать Открыть подменю редактирования
    🔗 Подключить реального ученика Только для виртуальных — ввести код реального ученика для привязки
    ✏️ Редактировать данные Только для виртуальных — изменить имя, фамилию и другие поля

    ✏️ Подменю редактирования ученика

    Из карточки ученика по кнопке ✏️ Редактировать открывается подменю:

    Кнопка Описание
    📋 Договор с учеником Генерация и просмотр PDF-памятки с правилами работы
    💰 Стоимость Изменить почасовую ставку для ученика
    🔗 Ссылки Настроить ссылки на созвон и доску для каждого предмета
    📚 Предметы Добавить или убрать предметы у ученика

    📋 Договор с учеником

    Генерация PDF-памятки с правилами работы:

    1. Нажмите 📄 Сгенерировать новые договорённости.
    2. Система создаст документ на основе ваших настроек (частота занятий, стоимость, правила отмены).
    3. Используйте 📥 Скачать договорённости, чтобы получить файл.
    4. Кнопка 🔄 Обновить данные обновит документ актуальными параметрами.

    💰 Стоимость

    Установка почасовой ставки для ученика. Ставка используется для расчёта баланса и статистики.

    🔗 Ссылки

    Для каждого предмета можно задать:

    • 📞 Созвон — ссылка на видеозвонок (Zoom, Google Meet и т.д.)
    • 📋 Доска — ссылка на интерактивную доску (Miro и т.д.)

    Ссылки доступны ученику и родителю — они смогут открыть их одним нажатием.

    📚 Предметы

    Управление предметами ученика:

    • Нажмите на предмет с отметкой ✅, чтобы убрать его (данные архивируются, не удаляются).
    • Нажмите на предмет без отметки, чтобы добавить его.
    • Нельзя убрать единственный активный предмет — должен остаться хотя бы один.

    📊 Успеваемость

    Кнопка 📊 Успеваемость в карточке ученика открывает детальную статистику:

    • Количество проведённых занятий за месяц
    • Количество выданных и выполненных ДЗ
    • Средний балл по домашним заданиям
    • Накопленные Газики
    • Динамика прогресса по предметам

    ⚠️ Важно знать

    1. При удалении ученика (через ✏️ Редактировать → ❌) происходит архивация — данные сохраняются и могут быть восстановлены.
    2. Виртуального ученика можно подключить к реальному в любой момент без потери данных.
    3. Количество учеников ограничено тарифом: Базовый — 3, Обычный — 15, Премиум — 100.
    4. При превышении лимита тарифа лишние ученики помечаются 🔒 и становятся недоступны до повышения тарифа.
