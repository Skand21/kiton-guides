# KITON Для Родителей

**Добро пожаловать!** KITON — ваш помощник для контроля обучения ребёнка. Всё в одном месте: домашние задания, оплаты, прогресс. Больше не нужно спрашивать "Сделал ли ты домашку?" — просто откройте бота! 😊

---

## 🚀 Быстрый старт

### 1. Регистрация

1. Откройте [@tutor_auto_bot](http://t.me/tutor_auto_bot) в Telegram
2. Нажмите **«Запустить»** или `/start`
3. Выберите роль **👨‍👩‍👧 Родитель**
4. Заполните: Фамилия, Имя (кириллица), Телефон (+7XXXXXXXXXX)

### 2. Подключите ребёнка

1. Попросите ребёнка открыть **👤 Профиль** в боте
2. Он скопирует персональный код (например, `ABCD-1234-EFGH`)
3. В меню нажмите **➕ Добавить ученика**
4. Вставьте код — готово!

**💡 Совет:** Можно подключить несколько детей — все будут видны в одном меню.

## 📋 Главное меню

| Кнопка | Что внутри |
| --- | --- |
| **📚 Домашние задания** | Все задания детей, статусы, оценки |
| **💳 Оплаты** | Баланс по репетиторам, история, долги |
| **📈 Прогресс** | Статистика: выполненные задания, средний балл, Газики |
| **❓ Помощь** | Чат поддержки, база знаний |
| **👤 Профиль** | Редактирование данных, управление детьми |

**Индикаторы:**

- **📚 (5)** — 5 активных заданий
- **💳 🔴** — есть долг
- **💳 🟢** — всё в порядке
