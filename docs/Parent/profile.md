# Профиль

В разделе **«Профиль»** хранятся ваши данные как родителя.

![Профиль пользователя](../assets/images/profile-edit.png)

*В профиле можно проверить данные и при необходимости отредактировать их.*

---

## Редактирование данных

Нажмите на кнопку нужного поля, чтобы изменить:

- **Фамилия** — только кириллица, с большой буквы
- **Имя** — только кириллица, с большой буквы
- **Телефон** — формат +7XXXXXXXXXX

**⚠️ Важно:** Система проверяет формат. Если данные неверны, изменения не сохранятся.

---

💡 **Совет:** Заполните профиль полностью — репетиторам будет проще с вами связаться.
