# KITON Для Учеников

**KITON** — платформа, которая помогает организовать обучение с репетитором: домашние задания, оплаты, прогресс и мотивация через систему баллов «Газики».

---

## 🚀 Старт работы с платформой

### 1. Регистрация

1. Откройте бота [KITON](http://t.me/tutor_auto_bot) в Telegram по [@tutor_auto_bot](http://t.me/tutor_auto_bot) и нажмите «Запустить» или `/start`.
2. Выберите роль «👨🎓 Ученик».
3. Заполните профиль:
    - Фамилия (только кириллица, с большой буквы)
    - Имя (только кириллица, с большой буквы)
    - Класс обучения (число)
    - Школа
    - Город
    - Телефон (формат +7XXXXXXXXXX)
4. После регистрации вы получите **персональный код** (например, `ABCD-1234-EFGH`).

**💡 Важно:** Сохраните свой персональный код! Он нужен для подключения репетитора или родителя.

### 2. Подключение к репетитору

Есть **2 способа** подключиться к репетитору:

### 🔗 По ссылке (рекомендуется)

1. Репетитор отправляет вам персональную ссылку.
2. Перейдите по ссылке — вы автоматически подключитесь к репетитору.
3. Если вы ещё не зарегистрированы, сначала пройдите регистрацию, затем подключение произойдёт автоматически.

### 🔑 По персональному коду

1. Репетитор просит ваш персональный код.
2. Найдите код в разделе **«👤 Профиль»**.
3. Отправьте код репетитору.
4. Репетитор введёт код в систему — вы подключитесь.

## 📋 Главное меню бота

| Кнопка | Что внутри |
| --- | --- |
| **📅 Занятия** | Расписание по каждому репетитору, история и детали прошедших занятий |
| **📚 Домашние задания** | Список всех заданий от репетиторов, статусы, сдача решений |
| **💳 Оплаты** | Баланс по каждому репетитору, история оплат, долги |
| **🎮 Газики** | Ваши баллы, доступные награды, обмен на скидки и бесплатные занятия |
| **👤 Профиль** | Персональный код, редактирование данных |
| **❓ Помощь** | Быстрые инструкции и поддержка |

> **Примечание:** Если вы состоите в учебной группе (функция тарифа «Эксперт» репетитора), в меню также появится кнопка **«👥 Мои группы»** для просмотра расписания и участников группы.

---

### 🔔 Индикаторы в меню

- **📚 Домашние задания** — число показывает количество активных заданий
- **💳 Оплаты** — 🔴 красный индикатор появляется при наличии долга
- **🎮 Газики** — число показывает общее количество накопленных баллов
