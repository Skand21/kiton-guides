# Групповые занятия

Раздел **«Группы»** позволяет вести занятия с несколькими учениками одновременно. Доступно на тарифе **Эксперт**.

---

## 📋 Список групп

Кнопка **«👥 Группы»** в меню показывает все ваши группы. Каждая группа отображается с названием, предметом и количеством участников.

Вверху — активные группы, внизу — кнопка **«📦 Архив групп»** для просмотра скрытых.

---

## ➕ Создание группы

1. Нажмите **«➕ Создать группу»**
2. Введите **название** (например, «Математика 9 класс» или «ЕГЭ Физика»)
3. Выберите или введите **предмет**
4. Введите **стоимость за час для одного ученика** (в рублях)
5. Выберите **длительность** занятия (30, 45, 60, 90 или 120 минут)

Группа создана. Теперь добавьте учеников и настройте расписание.

---

## 👥 Участники

### Добавление

Нажмите **«➕ Добавить участника»** → выберите ученика из вашего списка. Можно задать индивидуальную стоимость для конкретного ученика (отличную от стоимости группы).

### Удаление

Нажмите на имя участника → **«❌ Удалить из группы»**. Ученик перестаёт получать уведомления и домашние задания группы.

---

## 📅 Расписание группы

### Добавление слота

1. Нажмите **«📅 Расписание»** в карточке группы → **«➕ Добавить слот»**
2. Выберите тип:
    - **🔄 Регулярное** — повторяется каждую неделю. Выберите день недели → введите время (ЧЧ:ММ) → выберите длительность
    - **1️⃣ Разовое** — одно занятие. Выберите дату в календаре → введите время → выберите длительность
3. Если время пересекается с другим занятием, бот покажет предупреждение

### Удаление слота

Нажмите на слот → **«❌ Удалить»** → подтвердите. Все будущие занятия по этому слоту отменяются автоматически.

---

## 📝 Отчёт о групповом занятии

После завершения группового занятия:

1. Отметьте **посещаемость** — чекбокс для каждого ученика (☐ / ☑)
2. Задайте **домашнее задание** — одно ДЗ на всю группу
3. Оцените работу каждого ученика — индивидуальные Газики

Уведомления уходят каждому участнику и их родителям.

---

## 📚 Групповые домашние задания

Домашнее задание создаётся один раз на всю группу, но каждый ученик сдаёт **своё решение** и получает **индивидуальную оценку**.

Создать ДЗ можно:

- Из отчёта о занятии (после перезаписи)
- Кнопкой **«📚 ДЗ»** в карточке группы
- Из раздела **«📚 Домашние задания»** в главном меню

---

## 🗑️ Архивирование

Нажмите **«🗑️ Архивировать группу»** в карточке группы. Архивированная группа перестаёт генерировать занятия и скрывается из основного списка.

Восстановить можно в любой момент: **«📦 Архив групп»** → выберите группу → **«♻️ Восстановить»**.

---

## ⚠️ Важно знать

- Групповые занятия доступны только на тарифе **Эксперт**
- Стоимость можно задать индивидуально для каждого участника
- Ссылки на видеозвонок и доску задаются отдельно для группы (аналогично индивидуальным ученикам)
- Архивирование не удаляет данные — история занятий и ДЗ сохраняется
