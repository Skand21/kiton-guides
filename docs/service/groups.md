# Группы

Группы нужны для занятий, где один репетитор работает сразу с несколькими учениками.

## Когда использовать группы

- Интенсивы.
- Мини-группы.
- Разборы пробников.
- Регулярные занятия с несколькими участниками.

## Что важно проверить

- У каждого участника есть карточка ученика.
- В группе указан предмет.
- В расписании создан групповой слот.
- После занятия корректно отображаются статус, оплата и ДЗ.

Если группа используется только как временный интенсив, после окончания ее можно оставить в истории и не удалять данные.
