# Партнерка

Партнерская программа нужна, чтобы приглашать новых репетиторов и видеть, кто зарегистрировался по вашей ссылке.

## Как работает реферальная ссылка

1. Репетитор получает свою ссылку или код.
2. Новый пользователь переходит по ссылке на лендинг или в сервис.
3. Код должен сохраниться при регистрации.
4. После регистрации связь отображается в партнерской базе.

## Что проверить

- Ссылка открывается.
- Код не теряется при переходе.
- Новый пользователь видит обычную регистрацию.
- После регистрации реферал попал в список партнерки.
