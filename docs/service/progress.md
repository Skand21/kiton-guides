# Прогресс и пробники

Раздел "Прогресс" нужен, чтобы видеть не просто список занятий, а динамику подготовки.

## Общий вид

В общем виде пробники отображаются как таблица: пробники идут по колонкам, задания - по строкам. Это похоже на привычную таблицу, где видно, какие задания ученик стабильно решает, а какие проваливает.

## Конкретный пробник

В конкретном пробнике можно смотреть дату, предмет, тип экзамена, первичные и вторичные баллы. В будущем сюда можно прикреплять файл варианта, чтобы ученик видел, по какому материалу шла проверка.

## Баллы

Первичные баллы считаются по выполненным заданиям. Вторичные баллы рассчитываются по внутренней шкале выбранного экзамена и предмета, если такая шкала есть.

!!! note "Текущий фокус"
    В первой версии прогресс ориентирован на ЕГЭ-сценарии. Если нужен другой экзамен или предмет, его лучше добавлять отдельной настройкой.
