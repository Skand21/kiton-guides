# Регистрация

## Репетитор

1. Откройте веб-сервис КИТОН.
2. Нажмите регистрацию или вход через удобный способ.
3. Выберите роль "Репетитор".
4. Заполните имя, фамилию, предметы и контактные данные.
5. После входа добавьте первого ученика.

## Ученик

Ученик может зайти в веб-кабинет через ссылку или подключиться через Telegram. Если ученик регистрируется сам, он выбирает роль "Ученик" и заполняет базовые данные.

## Родитель

Родитель подключается к ребенку через приглашение или код. После подключения он видит только данные своего ребенка.

!!! tip "Если регистрация идет по реферальной ссылке"
    Ссылка должна сохраниться при переходе с лендинга в сервис или Telegram-бот. Если код не подтянулся, его можно ввести вручную.
