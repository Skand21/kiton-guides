# Роли и доступы

В КИТОН есть три основные роли: репетитор, ученик и родитель. У каждой роли свой кабинет и свой набор действий.

## Репетитор

Репетитор управляет учениками, занятиями, ДЗ, оплатами, прогрессом и уведомлениями. Это главный рабочий кабинет.

## Ученик

Ученик видит свои занятия, активные и прошлые домашние задания, Газики, баланс и прогресс.

## Родитель

Родитель видит информацию по ребенку: занятия, домашние задания, оплату, баланс и прогресс. Родитель не редактирует расписание и не меняет данные репетитора.

## Telegram и веб-кабинет

Telegram нужен для быстрых уведомлений и сдачи ДЗ. Веб-кабинет нужен для полноценного управления: там удобнее смотреть расписание, редактировать задания, проверять оплаты и вести прогресс.
