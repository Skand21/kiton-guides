## Проверка соответствия продукту

При публикации убедиться:

- Нигде не обещаются отдельные кошельки и переносы между ними.
- Не описывается смена пароля (вход только по Telegram).
- Абонементы упомянуты только как «в KITON учёт по факту, абонементов нет».
- Навигация между разделами (README и ссылки) ведёт на существующие страницы.
- Тон и объём соответствуют [Правилам написания контента](CONTENT_RULES.md).
